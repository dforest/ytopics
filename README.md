# Ytopics

View yahoo(.co.jp) topics on your terminal.

## Installation

Add this line to your application's Gemfile:

    gem 'ytopics'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ytopics

## Usage

    $ ytopics
